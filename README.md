# HacknetIRCEnhancements

IRC 翻页 / IRC Paging

本功能来自 TAXCore，分离为独立模组。 / This feature is extracted from TAXCore as a standalone mod.

---

TAXCore 翻页功能的标准配置：
/ Standard TAXCore scrolling controls:

| 按键 / Key | 功能 / Action |
|------------|---------------|
| ↑ / ↓ | 逐行滚动 / Scroll one line |
| PageUp / PageDown | 翻页滚动 / Scroll one page |
| Home | 跳至最旧 / Jump to oldest |
| End | 跳至最新 / Jump to latest |

---

若已安装 TAXCore，此模组会自动跳过，不重复加载。
/ If TAXCore is installed, this mod skips loading automatically.
